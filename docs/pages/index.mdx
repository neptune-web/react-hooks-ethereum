# wagmi

`wagmi` is a React Hooks library for Ethereum, built on [**ethers.js**](https://github.com/ethers-io/ethers.js).

## Features

- Hooks for connecting wallets, signing messages, sending transactions, etc.
- TypeScript ready
- Zero-dependencies
- MIT License
