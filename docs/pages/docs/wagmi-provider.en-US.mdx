# WagmiProvider

The `WagmiProvider` manages configuration for all hooks using [React Context](https://reactjs.org/docs/context.html) (not to be confused with an [ethers Provider](https://docs.ethers.io/v5/api/providers/) of course).

```tsx
import { WagmiProvider } from 'wagmi'
```

## Usage

```tsx
import { WagmiProvider } from 'wagmi'

function App() {
  return (
    <WagmiProvider>
      <YourRoutes />
    </WagmiProvider>
  )
}
```

## Configuration

### client (optional)

A wagmi [`Client`](/docs/client) instance that consists of configuration options. Defaults to `createClient()`.

```tsx {4-9,13}
import { providers } from 'ethers'
import { WagmiProvider, createClient } from 'wagmi'

const client = createClient({
  autoConnect: true,
  provider(config) {
    return new providers.AlchemyProvider(config.chainId, 'Your Alchemy ID')
  },
})

function App() {
  return (
    <WagmiProvider client={client}>
      <YourRoutes />
    </WagmiProvider>
  )
}
```
